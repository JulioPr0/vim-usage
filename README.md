## vim-usage

```
s untuk surround ( plugin vim-surround )

i untuk inner

d untuk delete

c untuk change

t untuk till ( sampai sebelum karakter tertentu )

f untuk find ( sampai karakter tertentu )

y untuk yank ( copy )

p untuk paste ( paste )

w untuk word ( kata di depan )

b untuk begin ( awal karakter dalam kata )

e untuk end ( akhir karakter dalam kata )

I untuk insert The First character of this line

A untuk append The End character of this line

cc untuk change this line

dd untuk delete this line

c3w untuk change 3 words

c3b untuk change 3 words ( backward )

d3w untuk delete 3 words

d3b untuk delete 3 words ( backward )

d3} untuk delete next paragraph

d3{ untuk delete previous paragraph

dtf untuk delete till f

dff untuk delete find f

\\w untuk cursor to word ( easymotion )

\\b untuk cursor to begin ( easymotion )

\\e untuk cursor to end ( easymotion )

\\f untuk cursor to founded char ( easymotion )

\\cc untuk comment this ( NERDcommenter )

\\cu untuk uncomment this ( NERDcommenter )

ctrl-u untuk Up

ctrl-d untuk down

H untuk high

M untuk medium

L untuk low

csw'" untuk change surround words ' to "

ci' untuk change inner '

cit untuk change inner tag

/ untuk search pattern

? untuk search pattern ( backward )

n untuk next search pattern

N untuk previous search pattern ( backward )

; untuk next char ( untuk f atau t )

, untuk previous char ( untuk f atau t )
```
